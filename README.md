# Career Guru
 
### Run command git clone (repo link)

## To start front-end

### cd Client 
### cd main 
### npm install
### npm start or npm run start

# To start backend

## To start course_backend

#### cd Server
#### cd course_backend
#### npm install 
#### npm run server

## To start text_chat_backend

#### cd Server
#### cd text_chat_backend
#### npm install 
#### npm start

## To start vid_chat_backend

#### cd Server
#### cd vid_chat_backend
#### npm install 
#### npm run server
